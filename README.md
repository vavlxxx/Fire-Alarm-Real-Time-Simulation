# Система пожарной сигнализации (PyQt6, имитация СРВ)

Приложение моделирует работу системы реального времени для пожарной сигнализации в учебном проекте.

## Ключевые особенности
- Полностью переписанный интерфейс на `PyQt6` (без `tkinter`).
- Интерактивная карта помещения на `QGraphicsView`:
  - масштабирование колесом мыши (как в картах);
  - панорамирование перетаскиванием;
  - выбор зоны ЛКМ;
  - контекстное меню зоны по ПКМ.
- Три состояния зоны:
  - `Норма`;
  - `Избыточный дым`;
  - `Избыточная температура (пожар)`.
- Автоматика приводов (по флажку):
  - при избыточном дыме включается дымоудаление;
  - при пожаре включаются спринклеры и дымоудаление;
  - после нормализации состояния приводы отключаются.
- Главный экран:
  - слева карта;
  - справа сверху панель управления;
  - справа снизу журнал событий.
- Вкладка `Данные`:
  - таблица зон;
  - графики температуры и дыма выбранной зоны (`pyqtgraph`).

## Архитектура
- `app.py` — вход в приложение (`QApplication`, запуск окна).
- `ui.py` — интерфейс PyQt6, карта, таблица, журнал, сценарии управления.
- `simulation.py` — модель зон, датчики, переходы состояний, автоматика.
- `charts.py` — графики на `pyqtgraph`.
- `strings_ru.py` — русские строки интерфейса и сообщений.
- `assets/zones_layout.json` — разметка зон на плане.

## Установка и запуск
```bash
pip install PyQt6 pyqtgraph
python app.py
```

## Формат разметки зон (`assets/zones_layout.json`)
```json
{
  "image": "images/Одноэтажный комплекс Одуванчик.png",
  "zones": [
    {
      "id": 1,
      "name": "Зона 1",
      "polygons": [[[0.0446, 0.0787], [0.2285, 0.0787], [0.2285, 0.4595], [0.0446, 0.4595]]],
      "label": [0.1366, 0.2691]
    }
  ]
}
```

### Правила
- `polygons` — набор полигонов зоны.
- Координаты нормализованы (`0..1`) относительно размера изображения.
- Каждая зона имеет стабильный `id`.

## Логика симуляции
Пороговые значения:
- `Избыточный дым`: `smoke >= 45 ppm`.
- `Пожар`: `temp >= 55 °C`.
- Возврат к норме по гистерезису: `temp < 45 °C` и `smoke < 35 ppm`.

## Соответствие этапам лабораторной работы
- Этап 1: дружественный динамический интерфейс оператора.
- Этап 2: имитация процесса с таблицей и графиками (оси и единицы измерения).
- Этап 3: имитация показаний датчиков.
- Этап 4: имитация приводов (спринклеры, дымоудаление).
- Этап 5: моделирование аварийных ситуаций и возврат к нормальному состоянию.
