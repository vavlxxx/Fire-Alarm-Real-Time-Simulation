# Система пожарной сигнализации (имитация СРВ)

## Запуск
```bash
python app.py
```

## Что есть в приложении
- Вкладка `Главный экран`:
  - карта помещения слева (около 70% верхней области);
  - графики выбранной зоны справа (около 30% верхней области);
  - панель управления внизу.
- Вкладка `Данные`: объединённые таблица зон и журнал событий.
- Все подписи интерфейса, журнал и справка отображаются сразу на русском языке (без транслитерации).

## Управление
- ЛКМ по зоне на плане: выбрать зону.
- ПКМ по зоне: контекстные действия (`Пожар`, `Задымление`, `Ручной извещатель`, `Неисправность`, сбросы, приводы).
- Масштаб карты: колесо мыши.
- Перемещение карты: средняя кнопка мыши (зажать и тянуть) или `Shift + ЛКМ`.
- Кнопки в блоке управления:
  - быстрые сценарии аварий;
  - автоматизация (`Автосценарии`, `Автовосстановление`);
  - приводы (`Спринклеры`, `Дымоудаление`);
  - операторские команды (`Квитировать`, `Отключить оповещатели`, `Сброс системы`).

## Зоны и карта
Разметка зон хранится в `assets/zones_layout.json`.

Формат:
- `image`: путь к изображению плана (относительно `assets`), например `images/Одноэтажный комплекс Одуванчик.png`.
- `zones`: список зон.
- Для каждой зоны:
  - `id`: номер зоны;
  - `name`: имя зоны;
  - `polygons`: массив полигонов, координаты нормализованы в диапазоне `0..1`;
  - `label`: позиция подписи зоны (`x`, `y`) в `0..1`.

## Звуки
- Циклическая тревога: `assets/sounds/alarm.mp3` (также поддерживается `assets/sounds/alam.mp3`).
- Короткий сигнал при переключении зоны: `assets/sounds/beep.mp3`.

## Индикация тревоги
- При тревоге зоны мигают на плане.
- Заголовок и системные надписи сверху мигают красным, чтобы тревогу было видно на любой вкладке.
